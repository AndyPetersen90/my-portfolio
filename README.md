# Andrew Petersen Portfolio

A personal portfolio site for Andrew Petersen, focused on selected full-stack web work, professional experience, skills, and contact information.

## Built With

- React
- TypeScript
- Vite
- Tailwind CSS
- Vercel Analytics

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Run quality checks:

```bash
npm run lint
npm run build
```

## Content Focus

The site is intentionally organized around proof of work first:

- Clear hero message
- Featured project case studies
- Resume and experience summary
- Grouped technical skills
- Short personal background
- Direct contact options
